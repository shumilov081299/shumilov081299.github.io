03 · QUICK TIPS

Quick tips.

TIP · 01

Assign three shortcuts

Each mode is a separate command: All Notes, Same Pitch Any, Same Pitch Exact. Assign a key to each.

TIP · 02

Use Bars display mode

For best accuracy with non-standard time signatures (3/4, 5/4, 7/4), set your transport display to Bars.

TIP · 03

Works with humanized MIDI

No need to quantize first. The script handles humanized notes up to ±50 ticks.
