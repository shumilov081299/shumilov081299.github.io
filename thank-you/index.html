Getting started

1
Download the .package file from your email
2
Copy to your Scripts folder:
Windows:
C:\Program Files\PreSonus\Studio One 6\Scripts\
C:\Program Files\PreSonus\Studio One 7\Scripts\
C:\Program Files\Fender\Studio Pro 8\Scripts\
macOS: Right-click .app → Show Package Contents → Contents/Scripts/
3
Restart your DAW completely
4
Keyboard Shortcuts → search "Select Equal Subpositions" → assign shortcuts to all three modes
5
Open Piano Roll, select a note, press your shortcut — done!

Quick tips

Assign three different shortcuts

Each mode is a separate command: All Notes, Same Pitch Any, Same Pitch Exact. Assign a key to each for fast switching.

Use Bars display mode

For best accuracy with non-standard time signatures (3/4, 5/4, 7/4), set your transport display to Bars.

Works with humanized MIDI

No need to quantize first. The script handles humanized notes up to +/-50 ticks automatically.
