Installation Guide

Windows

Copy the .package file to your Scripts folder:
C:\Program Files\PreSonus\Studio One 6\Scripts\
C:\Program Files\PreSonus\Studio One 7\Scripts\
C:\Program Files\Fender\Studio Pro 8\Scripts\
Restart your DAW. Go to Keyboard Shortcuts and assign keys.

macOS

Right-click the .app → Show Package Contents → Contents/Scripts/
/Applications/Studio One 6.app/Contents/Scripts/
/Applications/Studio One 7.app/Contents/Scripts/
/Applications/Fender Studio Pro.app/Contents/Scripts/
Restart your DAW. Go to Keyboard Shortcuts and assign keys.

FAQ

Does it work with humanized MIDI?

Yes. Smart tolerance catches humanized notes up to +/-50 ticks (Studio One's default humanize range).

What about time signature changes?

The script detects time signature at the selected note's position. Bars display mode gives best accuracy for non-standard signatures like 3/4 or 7/4.

Can I use it on multiple machines?

Yes. Install on all computers you personally own and use.

Which editions are supported?

Studio One Professional and Artist editions that support scripts.

Will I get updates?

Yes, all future updates are included at no extra cost.

The script doesn't appear in Keyboard Shortcuts

Make sure the .package file is in the correct Scripts folder and you've restarted the DAW completely (not just the project).

Nothing happens when I run the script

Make sure you have at least one note selected in the Piano Roll before running the shortcut.
