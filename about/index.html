About shmlv

One person. Real tools. No filler.

shmlv is a solo operation based in Ukraine. I make workflow tools for music producers — scripts, plugins, and utilities that solve specific problems in Studio One and Fender Studio Pro.

I started because I kept running into the same friction in my own sessions. Features that exist in Logic Pro or Cubase but not in Studio One. Tasks that take 50 clicks when they should take 1. So I built the tools myself.

Every tool I release is something I use in my own work. If it saves me time, it'll save you time. No bloated suites, no subscription traps. Just focused utilities that do one job well.

The name? shmlv = Shumilov, minus the vowels. Keep it short, keep it sharp.
