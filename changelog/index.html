Select Equal Subpositions v1.0

APRIL 2026 · INITIAL RELEASE
  • Initial public release.
  • Ships three commands: All Notes, Same Pitch Any, Same Pitch Exact.
  • Supports Studio One 6, Studio One 7, and Fender Studio Pro.
  • Supports Windows, macOS Intel, and macOS Apple Silicon.
  • Includes humanize-aware matching up to ±50 ticks.
Updates

v1.x updates for Select Equal Subpositions are included at no extra cost. Future major versions may be separate products if the scope changes.
