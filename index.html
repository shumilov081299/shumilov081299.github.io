FAQ

Does it work with humanized MIDI?

Yes. The script quantizes anchor positions to the nearest sixteenth note grid and uses smart tolerance to catch humanized notes up to +/-50 ticks.

What if my song changes time signature?

The script detects the time signature at the selected note's position. Pro tip: Bars display mode gives the best accuracy for non-standard time signatures like 3/4 or 7/4.
