How it works

Three focused modes.

One package. Three assignable shortcuts. Each does exactly one job — precisely.

MODE 01

All Notes

Select a note — instantly select every note at that same beat position in every bar, regardless of pitch. Built for velocity sweeps, muting downbeats, and bulk transposition.

MODE 02

Same Pitch — Any

Finds every matching pitch at that beat position across all bars. Each pitch is searched independently — if it exists somewhere, it gets selected. Ideal for arpeggios, repeating licks, motif work.

MODE 03

Same Pitch — Exact

Selects only bars where the full selected pitch set appears together at the same beat position: every repeated C-major hit on beat 1, every identical voicing, every matching accent.

Features

Key features.

Built for real MIDI — humanized notes, common time signatures, and practical Studio One display modes.

Humanize-Aware

Matches humanized notes within ±50 ticks of the target position.

Auto Time Signature

Detects 2/4, 3/4, 4/4, 5/4, 6/4, 7/4, 8/4 automatically from the DAW transport.

Display Modes

Works in common transport display modes. For unusual time signatures, use Bars display mode for best accuracy.

Safe & Reliable

If no match is found, your original note selection is restored.

Multi-Note Selection

Select multiple notes on different beats — finds all matching positions independently.

One Package

Single .package file installs all three modes. Assign separate keyboard shortcuts to each.

FAQ

FAQ.

Does it work with humanized MIDI?

Yes. Smart tolerance catches humanized notes up to ±50 ticks — Studio One's default humanize range.

What about time signature changes?

The script detects time signature at the selected note's position. Bars display mode gives best accuracy for non-standard signatures.
