02 · HOW IT WORKS

Three powerful modes.

One package. Three assignable shortcuts. Each does exactly one job — precisely.

MODE 01

All Notes

Select a note — instantly select every note at that same beat position in every bar, regardless of pitch. Just like Logic Pro. Great for velocity sweeps, muting downbeats, bulk transposition.

MODE 02

Same Pitch — Any

Finds every matching pitch at that beat position across all bars. Each pitch is searched independently — if it exists somewhere, it gets selected. Ideal for arpeggios, repeating licks, motif work.

MODE 03

Same Pitch — Exact

Finds only bars where all selected pitches appear together at that position. Missing even one note — the bar is skipped. Precision tool for surgical MIDI editing: every C major chord in a song, every 3-note voicing, every identical hit.
