DOCS · SELECT EQUAL SUBPOSITIONS

Install. Assign. Edit.

The practical manual for Select Equal Subpositions: what to install, where to put it, how each command behaves, and what to check if something does not appear.

Requirements Install paths Shortcuts Modes Troubleshooting Support

Requirements

  • Studio One 6 or Studio One 7 with scripting support.
  • Fender Studio Pro with scripting support.
  • Windows or macOS, Intel or Apple Silicon.
  • A MIDI editor / Piano Roll selection. The command works from selected notes.

Install paths

Windows

C:\Program Files\PreSonus\Studio One 6\Scripts\ C:\Program Files\PreSonus\Studio One 7\Scripts\ C:\Program Files\Fender\Studio\Scripts\
This is the manual host Scripts folder install route. You may need admin permission on Windows, and major DAW updates can overwrite unmanaged app folders, so keep a backup of the downloaded .package file.

macOS

Right-click the app, choose Show Package Contents, then open:

Contents/Scripts/

Restart the DAW completely after copying the package. Closing only the song is not enough.

Assign shortcuts

Open Keyboard Shortcuts and search for Select Equal Subpositions. Assign separate shortcuts to all three commands.

Recommended setup: keep the three shortcuts physically close on the keyboard. The tool is fastest when the modes feel like one small editing cluster.

Modes

All Notes

Selects every note at the same beat position across bars, regardless of pitch. Best for downbeats, velocity sweeps, repeated hits, and broad edits.

Same Pitch — Any

Searches matching pitch at the same beat position. Each selected pitch is searched independently.

Same Pitch — Exact

Selects only bars where the full selected pitch set appears together. Best for chord voicings and exact repeated hits.

Troubleshooting

The command does not appear

Check that the .package file is inside the correct Scripts folder, then fully restart the DAW.

The shortcut runs but nothing happens

Select at least one MIDI note in the Piano Roll before running the command.

The wrong notes are selected

Use Bars display mode for best accuracy with unusual time signatures. If the part is heavily humanized beyond ±50 ticks, quantize an anchor note or email support with a short example.

Support

Email support@shmlv.com with your purchase email, DAW version, operating system, install path, Keyboard Shortcuts search term, and a short description of the issue.
